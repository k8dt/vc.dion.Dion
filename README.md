# Dion

This repo hosts the flatpak wrapper for the desktop version of [Dion video conferencing service with a hybrid cloud infrastructure.](https://diongo.ru/) and does not contain any Dion-licensed binary code. Instead, it is downloaded and converted to flatpak format on the fly from the official Dion repository by running the flatpak install/update commands.

**NOTICE: This wrapper is not verified by, affiliated with, or supported by dion.vc.**

## Issue reporting

**Please only report issues in this repo that are specific to the flatpak wrapper.**

Issues that can be replicated in another release should be reported via mailto:support@diongo.ru.
Make sure, that the reported issue is **not** flatpak-related.
